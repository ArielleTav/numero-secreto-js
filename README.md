Jogo do Número Secreto
📌 Sobre o projeto

Jogo interativo desenvolvido com HTML, CSS e JavaScript, onde o usuário deve adivinhar um número secreto gerado aleatoriamente. O sistema fornece dicas durante as tentativas e exibe uma tela de sucesso ao acertar.

🚀 Funcionalidades
Geração de número aleatório
Dicas (maior ou menor)
Contador de tentativas
Feedback visual ao acertar
Interface estilizada com imagens
💻 Tecnologias utilizadas
HTML
CSS
JavaScript
